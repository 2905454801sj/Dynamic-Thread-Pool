Dynamic Thread Pool Management System

动态线程池管理系统

Overview

A Spring Boot application that provides dynamic thread pool management with real-time monitoring and alerting capabilities.

Features	

1.Dynamic Thread Pool: Configurable thread pool with runtime parameter adjustment

2.Real-time Monitoring: REST API endpoints for thread pool status monitoring

3.Alert System: Configurable threshold-based alerting with rejection tracking

4.Custom Thread Factory: Named thread creation with proper configuration

5.RESTful API: Easy integration with monitoring tools

Tech Stack

Java: JDK 17

Spring Boot: Web framework and dependency injection

Maven: Build and dependency management

ThreadPoolExecutor: Core thread pool implementation
