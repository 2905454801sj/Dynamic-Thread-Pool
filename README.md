Dynamic Thread Pool Management System
动态线程池管理系统
Overview
A Spring Boot application that provides dynamic thread pool management with real-time monitoring and alerting capabilities.

Features
    Dynamic Thread Pool: Configurable thread pool with runtime parameter adjustment
    Real-time Monitoring: REST API endpoints for thread pool status monitoring
    Alert System: Configurable threshold-based alerting with rejection tracking
    Custom Thread Factory: Named thread creation with proper configuration
    RESTful API: Easy integration with monitoring tools
Tech Stack
Java: JDK 17
Spring Boot: Web framework and dependency injection
Maven: Build and dependency management
ThreadPoolExecutor: Core thread pool implementation
